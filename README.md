# Multithreaded Chat Application

## Overview
A Java-based multithreaded chat application that allows multiple clients to connect to a server and exchange messages in real time. The project uses socket programming and multithreading to handle multiple client connections simultaneously.

## Features
- Multi-client communication
- Real-time message broadcasting
- Server-client architecture
- Concurrent client handling using threads
- Built using Java Socket Programming

## Technologies Used
- Java
- Multithreading
- Socket Programming
- Eclipse IDE
- Git & GitHub

## How to Run

### Start the Server
1. Open the project in Eclipse.
2. Run `Server.java`.
3. The server will start and wait for client connections.

### Start the Client
1. Run `Client.java`.
2. Connect to the server.
3. Open multiple client instances to test real-time communication.
